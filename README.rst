dot *Beta*
================

A git inspired CLI to manage your dotfiles

Please ★ this repo if you found it useful ★ ★ ★


Features
--------

-  Sync with git
-  Automatic symlinking


Installation
------------

.. code:: sh

       sudo pip install dot
       dot clone


Usage
------------

Run the following every time you make changes

.. code:: sh

       dot sync
